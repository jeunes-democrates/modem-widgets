Ajoute facilement des liens utiles du Mouvement D&eacute;mocrate.

== Description ==



= Intro = 

Avec ce plugin les utilisateurs pourront simplement ajouter les liens utiles du Mouvement D&eacute;mocrate sous forme de widget.

Le plugin propose pour le moment 4 widgets :

*	Widget "Nous soutenir" : logo de la boutique + Lien vers la boutique
*	Widget "Adhérer" : image d'une carte d'adhérent + Lien personnalisé (et vers le formulaire en ligne si champs vide)
*	Widget "manifeste jeune" : image + lien vers le manifeste optimiste jdem
*	Widget "Livret d'accueil" : image + lien vers le livret d'accueil

== Installation ==

Extraire le fichier .tar.gz et le placer dans le dossier wp-content/plugins/ de votre installation WordPress, puis, l'activer sur la page des extensions install&eacute;es.
